Opsin Helper

A small background service required for live metrics.

What is Opsin Helper?

Opsin runs entirely in your browser and cannot read your system hardware directly. Opsin Helper is a small, lightweight background service that reads your CPU, GPU, and RAM and shares it with Opsin over a private local connection — nothing ever leaves your machine.

Once installed it runs silently and starts automatically with Windows, so you always have live stats without any extra steps.

How to install:

  1. Click Download Installer below and save the file
  2. Run OpsinHelperInstaller.exe — if Windows SmartScreen appears, click "More info" then "Run anyway"
  3. The installer adds the helper to your Windows Startup folder so it launches at login
  4. Reload Opsin — metrics will appear in the status bar within a few seconds

The installer is safe. It places a single Python-based service that only listens on localhost:4821 — completely inaccessible from the internet. The full source code (opsin_helper.py) is included in the download folder so you can inspect exactly what it does.
